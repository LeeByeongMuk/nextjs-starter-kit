# Next Template

## Description

최신 Next.js 를 활용해 자체 로그인 및 게시판 기능을 구현해놓은 템플릿입니다.

- Node 20.x
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- React-Query
- React-Hook-Form

## Run the Development Server

#### next.js 실행 방법

```bash
npm install
cp .env.example .env
npm run dev
``` 

#### 도커 설치 후 실행 방법

https://www.docker.com/products/docker-desktop

```bash
npm install
cp .env.example .env
docker compose up -d
```
